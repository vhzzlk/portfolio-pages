API + UI para autenticação

Projeto enxuto composto por uma API Spring Boot (Maven) e um front-end limpo dedicado a autenticação, ideal para demonstrar fluxo de login completo.

  • API em Spring Boot com endpoints de sessão, hashing de credenciais e padrões REST bem definidos.
  • Repositórios separados garantem deploy independente e facilitam auditoria de código.
  • Interface minimalista que prioriza foco, estados de erro claros e responsividade.
